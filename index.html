Intelligent PDF Converter (OCR + Pandoc) GitHub

  • Engineered a desktop application with a CustomTkinter GUI that converts scanned PDFs into reformatted, text-searchable PDF documents.
  • Employs Tesseract OCR via PyMuPDF to accurately extract text from image-based PDF pages, using multi-processing for parallel execution and enhanced performance.
  • Applies regex-based heuristic algorithms to identify and format headings, chapters, and paragraphs to create a structured Markdown output.
  • Uses the Pandoc engine to convert the generated Markdown into a clean, professionally formatted PDF with user-configurable options.
Technologies: Python, CustomTkinter, Tesseract (pytesseract), PyMuPDF (fitz), Pandoc, Multiprocessing, Regex.
