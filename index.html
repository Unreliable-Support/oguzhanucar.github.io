AI Copy Editor & Cliche Hunter

Advanced Editorial Tool with Persona-Driven AI

Project Overview

An advanced, full-stack editorial tool designed to cleanse text of cliches, enforce specific stylistic guidelines, and perform deep redaction using Google Gemini API. It allows editors to define "forbidden phrases" and automatically generates professional alternatives while preserving context.

Key Features & Architecture

  • Multi-Mode AI Analysis: Features specific modes for "Cliche Cleaning" (replacing banned words) and "Redaction" (improving flow, grammar, and tone) using distinct, expert-crafted system prompts.
  • Interactive Diff & Sidebar: A sophisticated UI that highlights changes in real-time. Users can review, accept, or reject specific AI suggestions via a side-by-side comparison panel.
  • "Ask Editor" & Custom Prompts: Includes a "Redactor Persona" mode where users can ask specific questions about the text or provide custom rewriting instructions.
  • Full File Processing: Supports direct processing of .txt and .pptx files. It can deconstruct PowerPoint presentations, edit the text content, and rebuild the file for download.
  • Global Comments System: Allows editors to leave persistent comments on specific sentences or revisions, creating a collaborative workspace feel.

Psych Cleaner

AI-Powered Journaling & Therapy Platform

Project Overview

Psych Cleaner is a comprehensive, full-stack mental wellness application designed from the ground up to serve as a private, AI-powered companion for psychological self-exploration and structured therapy. The platform integrates multiple evidence-based therapeutic modalities into a cohesive digital experience.

Key Features & Architecture

  • Core Journaling System: A robust free-form journaling module with rich tagging capabilities for categories, people, and recurring themes, allowing for deep contextual organization.
  • AI Co-pilot for Insight & Therapy: Leverages the Google Gemini API for a suite of intelligent features, including contextual analysis of entries, streaming therapy chat sessions, and an AI writing assistant.
  • Structured Therapy Journals: Implements digital versions of established therapeutic techniques like CBT Thought Records, Gratitude Journals, and Expressive Writing.
  • "Self-Discovery" Module: A unique, guided, multi-stage journey for deep self-reflection covering the user's past, present, and future.
  • Integrated Clinical Programs: Features digitized, interactive versions of renowned, evidence-based therapy programs for conditions like Insomnia (CBT-I) and Social Anxiety (CBT).

Flawless PDF Annotator

AI-Enhanced Research Tool

Project Overview

A sophisticated, feature-rich PDF reader and research environment developed from scratch using Python and PyQt6. This desktop application is designed for academic and professional use, integrating powerful annotation capabilities with a context-aware AI assistant to create a seamless workflow for deep document analysis.

Key Features & Architecture

  • High-Performance PDF Rendering: Utilizes the PyMuPDF (Fitz) library for fast and accurate page rendering, with performance optimizations like lazy loading and render caching.
  • Three-Pane Interface: A highly organized layout featuring a sidebar for navigation, a central PDF viewer with custom controls, and a dedicated AI assistant panel.
  • Advanced Annotation Suite: Supports precise, character-based text selection, multi-color highlighting, and persistent text notes directly to the PDF file.
  • Integrated AI Assistant (Gemini): A powerful AI panel provides contextual chat about the document content, along with customizable "Smart Actions" to summarize or analyze specific sections.
