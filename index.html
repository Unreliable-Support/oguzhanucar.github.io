Summarizer 3000: AI-Powered Full-Stack Research & Learning Environment YouTube Demo

Spearheaded the design and full-stack development of this project, transforming an initial desktop concept into a sophisticated, AI-driven web application designed to streamline the entire academic research, writing, and learning lifecycle. The platform integrates every step of the process—from initial content ingestion and drafting to final manuscript creation and exam preparation—within a single, cohesive environment.

Key Contributions & Features:
  • Full-Stack Web Architecture: Developed a robust backend using Flask and SQLAlchemy, featuring a RESTful API that serves both a dynamic, single-page web application and a companion Chrome extension. The frontend is built with vanilla JavaScript, HTML5, and Bootstrap to deliver a responsive and interactive user experience.
  • Advanced AI-Powered Writing Assistant: Engineered a multi-panel "Writing Assistant" module that actively guides the user's writing process. This includes a rich-text editor, contextual source management, conversational AI chat, and "Smart Features" for automated academic outlining and full-length paper generation based on user profiles and selected sources.
  • Personalized "Learn Anything" Module: Designed and implemented an innovative "Learn Anything" module where users can create persistent, database-backed "Learning Libraries." Users can curate sources, define learning goals, and engage in contextual, stateful conversations with an AI tutor that references the provided materials, personalizing the learning journey.
  • Automated Content Generation & Study Tools: Automated the creation of complex study materials, including multi-day study plans derived from LLM outputs, varied exam types (MCQ, essay), and themed PowerPoint presentations. This process involves sophisticated parsing of structured LLM responses and dynamically generating calendar events and linked materials.
  • Comprehensive Ecosystem & User Experience: Orchestrated a seamless user workflow, from multi-format content ingestion (PDFs, DOCX, YouTube transcripts, websites) and hierarchical project management to a full-featured dynamic calendar and an accompanying Chrome browser extension for on-the-fly interaction with web content.
Backend: Python, Flask, SQLAlchemy
Frontend: JavaScript (Vanilla), HTML5, CSS3, Bootstrap 5
AI & Data Processing: Google Generative AI (Gemini), Pandoc, PyPDF2, python-pptx, ebooklib
Database: SQLite
Tools & Ecosystem: Chrome Extensions (Side Panel API), Git

Note Harvester: Desktop Utility GitHub YouTube Demo

A robust desktop utility to instantly capture selected text from any application with a global hotkey. It organizes clippings into distinct notebooks, providing a seamless workflow for research and information gathering.

  • Global Hotkey Capture: Instantly saves any highlighted text from anywhere in the OS with a customizable hotkey (e.g., `++A`).
  • Crash-Proof Design: The capture mechanism temporarily pauses the hotkey listener during the copy operation and guarantees its restart, ensuring high stability and preventing conflicts.
  • Rich GUI for Organization: Features a clean Tkinter interface for creating and managing multiple notebooks, filtering notes by text or source, and merging multiple entries into one.
  • System Tray Integration: Runs unobtrusively in the background and can be minimized to the system tray for easy access without cluttering the desktop.
Technologies: Python, Tkinter (ttk), pynput, pygetwindow, pystray, pyperclip, JSON.
